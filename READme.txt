### Naming of plots:
"_detect": residual energies from a set of samples
"_s": source.photon()
"_fs": fake_source(... normal)
"_fsi": fake_source(...ideal)


format:
<source parameter: kvp_mmfilter>
_<source type: "s/fs/fsi">

if for ct_detect result of residual energies:
    _<materials>
    _<depth>
    _<plot type:"_detect"/nil>
